【训练题】字符串处理
  时间限制：1秒　　内存限制：64M
【问题描述】
 
　　输入四个字符串 a、b、c 和 d，其中字符串 d 是一个只含数字的字符的字符串，请你编程实现下列功能：

　　任务1：计算并输出四个字符串的长度和。

　　任务2：输出 a，b，c 这三个字符串中较大的一个。

　　任务3：查找并输出字母 'b' 在字符串 b 中最早出现的位置（字符串的第一个字符的位置是 0），如果不存在则输出 -1。

　　任务4：查找并输出字符串 b 在字符串 a 中最早出现的位置（字符串的第一个字符的位置是 0），如果不存在，则输出 -1。

　　任务5：交换字符串 a 和 b 后输出 a 和 b 。

　　任务6：反转字符串 c 后输出 c 。

　　任务7：把字符串 a 的子串 a[i]..a[j] 插入到字符串 b 的第 k 个位置，即b[k] 之后，然后输出b。

　　任务8：把数字字符串 d 转换为整数 x,然后输出 5*x 的值。

　　任务9：把字符串 c 连接到字符串 a 的后面，然后输出 a 。

　　任务10：找出字符串 a 中 ASCII 码最大的字符，并输出这个字符和它的ASCII码。

　　任务11：判断连接后的字符串 c 是否是回文串，是则输出'Yes'，否则输出'No'。

　　任务12：把字符串 a 中的所有小写字母修改成大写，把字符串 b 中所有大写字母修改成小写字母，然后输出修改后的 a 和 b 。

　　任务13：把字符串 a 中子串 a[i]..a[j] 删除，然后输出字符串 a 。

　　注意：上面的所有任务都是在前面的任务的基础继续往下做，例如：任务 7 的字符串 a 和字符串 b，是在任务 5 交换过后的 a 和 b，也就是说这里 a 是原来 b，这里的 b 是原来的 a 。其他任务也是这样。
 
【输入格式】
 
　　第 1 行：一个只包含大写字母、小写字母的字符串，表示a 。
　　第 2 行：一个只包含大写字母、小写字母的字符串，表示b 。。
　　第 3 行：一个不含空格的字符串，表示 c 。
　　第 4 行：一个只包含数字字符的字符串 d 。
　　第 5 行：包含三个整数 i、j 和 k  。
 
【输出格式】
 
　　共 13 行，分别对应 任务1..任务13 的输出结果 。
 
【输入样例】
 
ABCDEFabcde123
EFabc
aAbBcCcBbAa
8866
2 4 3

【样例解释】
 
　　“回文串”是一个正读和反读都一样的字符串，比如“pigip”或者“choohc”等等就是回文串。
 
【数据范围】
 
a,b,c 字符串包含的字符个数不超过100个，d 字符串长度不超过 8。
0 <= i <= j < strlen(b)。
0 <= k < strlen(a)
